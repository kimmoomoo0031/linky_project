## Linky バックエンド機能一覧（設計メモ）

このドキュメントは、Linky アプリのバックエンド機能一覧と、画面ベースで整理した仕様メモです。  
実装前の設計確認用 README です。

---

## 1. 認証・アカウント関連

- メール認証系
  - メールアドレス＋パスワードで新規登録
  - ログイン / ログアウト
  - メール認証（6桁コード）
  - パスワードリセット（メール送信 & 新パスワード設定）
  - 退会（アカウント削除）

- ソーシャルログイン
  - LINE ログイン
  - Google ログイン

- ゲスト利用
  - 非ログイン状態での投稿（仮ニックネーム＋パスワード）
  - 非ログイン状態でのコメント（仮ニックネーム＋パスワード）
  - ゲストは自分の投稿/コメントのみパスワードで削除・修正可能

---

## 2. ホーム・ラウンジ関連

- ホーム画面
  - 通知アイコン表示
  - ラウンジ検索バー
  - マイラウンジ一覧（日本生活、エンジニア など）
  - ベスト投稿リストの表示

- ラウンジ機能
  - ラウンジ一覧表示
  - ラウンジ詳細画面（そのラウンジの投稿一覧）

---

## 3. 投稿（記事）関連

- 投稿作成
  - ログインユーザー用投稿画面
  - ゲスト用投稿画面（ニックネーム＋パスワード）
  - タイトル / 本文 / 画像（最大 10 枚）登録（画像ファイル自体は S3 保存）

- 投稿閲覧
  - 投稿詳細画面
  - タイトル / 本文 / 画像表示
  - 作成者名 / 作成日時
  - 閲覧数表示
  - コメント数表示

- 投稿管理
  - 自分の投稿一覧（マイページから）
  - ログインユーザー: 自分の投稿のみ「修正」「削除」ボタン表示
  - ゲスト: 自分の投稿のみパスワード入力で「修正」「削除」可能
  - 管理者: 投稿管理一覧 / 投稿詳細から削除可能

- いいね / 「X」評価
  - 投稿に対する「おすすめ(いいね)」「X」
  - ボタン押下でカウントアップ
  - 二重押下時に「すでにおすすめ済み」等のメッセージ表示（取り消しは不可）
  - ベスト投稿判定（いいねが一定数以上で自動的にベスト扱い）
  - 「X」判断（「X」一定数超えると不適切な投稿だと判断し、管理者に自動通報）

- ベスト投稿
  - いいね数が閾値以上の投稿を「ベスト」に自動登録
  - ベストタブでベスト投稿一覧を表示

---

## 4. コメント関連

- コメント投稿
  - ログインユーザーコメント
  - ゲストコメント（ニックネーム＋パスワード）

- コメント表示
  - コメント一覧表示
  - 並び替え: 「登録順」 / 「新着順」

- コメント管理
  - 自分のコメントのみ削除可能
    - ログインユーザー: 即時削除
    - ゲスト: パスワード入力後に削除
  - コメント編集機能はなし（再投稿前提）

---

## 5. 検索機能

- 検索画面
  - キーワード検索
  - 検索条件:
    - タイトルで検索
    - 内容で検索
    - 投稿者で検索
  - 検索結果一覧表示
  - ラウンジ内検索（そのラウンジに絞った検索）

---

## 6. ベスト関連

- ベストタブ
  - ベスト投稿一覧表示
  - タイトル / 日時表示

---

## 7. マイページ・ユーザー設定

- マイページ（一般ユーザー）
  - プロフィール表示（ニックネーム、メールアドレス）
  - 自分の投稿一覧
  - プロフィール編集
  - 通知設定
  - ラウンジ申請への導線
  - ログアウト
  - 退会

- プロフィール編集
  - ニックネーム変更
  - メールアドレス表示
  - パスワード変更

- マイページ（非ログイン時）
  - 「マイページの利用にはログインが必要です」メッセージ
  -  ログインへの誘導

---

## 8. 通知・通知設定

- 通知一覧
  - 自分の投稿にコメントがついた時の通知
  - 自分のラウンジ申請が「承認 / 却下」された時の通知
  - 通知一覧画面で、通知リスト /　タイトル　/ 時刻を表示

- 通知設定
  - コメント通知 ON/OFF
  - ラウンジ申請結果通知 ON/OFF

- FCM 連携
  - ユーザーごとの FCM トークン保存
  - 新規コメント / ラウンジ申請結果時にプッシュ通知送信

---

## 9. ラウンジ申請 / 管理者機能

- 一般ユーザー用ラウンジ申請
  - ラウンジ申請画面（注意事項 + ラウンジ名 + 申請理由）
  - ラウンジ申請完了ダイアログ

- 管理者用機能
  - 管理者マイページ
    - 投稿管理一覧（全投稿を対象）
    - ラウンジ申請一覧（承認待ちリスト）
  - ラウンジ申請詳細から:
    - 「許可」ボタン → ラウンジ作成 + 申請ステータス更新 + 通知送信
    - 「お断り」ボタン → 却下 + 通知送信
  - 投稿管理詳細:
    - 投稿内容確認
    - 不適切な投稿の削除
